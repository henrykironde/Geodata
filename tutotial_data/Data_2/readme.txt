Datasets for R Geospatial Workshop, Nashville June 2012
=======================================================


The sample data used in the workshops comes from a variety of open data sources.

    UK Open Gazetteer places
    OpenStreetMap map tiles
    Geofabrik Shapefiles from OpenStreetMap data
    European Environment Agency Land Use Data
    NASA Shuttle Radar Topography Data

Please check the sources before any re-use.
